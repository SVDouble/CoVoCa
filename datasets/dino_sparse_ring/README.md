# dinoSparseRing

The Middlebury "dinoSparseRing" multi-view dataset: 16 photos of an
untextured ceramic dinosaur on a turntable, against a black background, with
known per-image camera intrinsics/extrinsics. Source:
<https://vision.middlebury.edu/mview/data/> (mirrored images used here via
<https://github.com/maximm8/VisualHull>). Provided by Steve Seitz, James
Diebel, Daniel Scharstein, Brian Curless, and Rick Szeliski for research and
educational use.

Of the 16 images, 15 have "good" silhouettes (one has a shadow artifact and
is skipped, per the dataset's own recommendation).

## Download

```bash
uv run --project covoca_toolkit --python 3.14 download-dino-sample
```

This writes to `local/datasets/dino_sparse_ring/`:

- `images/`: the 16 source photos.
- `masks/`: binary silhouette masks for the 15 "good" images, generated by
  thresholding at 0.19 of `[0, 1]`, dilating by 10px, and eroding by 7px (the
  recipe recommended by the dataset's `README.txt`).
- `calibration_result.yaml`: a `gs.calibration.result.v1` file synthesized
  from the dataset's `*_par.txt` camera parameters (`K`, `R`, `t` per image),
  with the dataset's world frame treated as covoca's "board" frame.

## Config

[`voxel_carving.yaml`](voxel_carving.yaml) carves the volume given by the
dataset's reference bounding box for the dino model (with a small padding
margin):

```text
(-0.061897, -0.018874, -0.057845) -- (0.010897, 0.068227, 0.015495)
```

`carving.outside_image_policy: ignore_view` is used because some images show
the object partially outside the field of view (per the dataset's README).

```bash
./build/debug/voxel_carve run --config datasets/dino_sparse_ring/voxel_carving.yaml
```
